
# Описание

JS библиотека для шифрования сообщений в EraChain сети.

# Подготовка

git clone https://github.com/avdeeviv/eracripto.git

yarn install (для варианта 2)

# Сборка

yarn build (для варианта 2)

# Использование

Вариант 1:
    Скачать папку /src.

Пример:
    src/index.html

Вариант 2:
    Скачать папку /dist

Пример:
    dist/example_webpack.html
